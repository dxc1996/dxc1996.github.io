js跨域
利用了script src没有同源限制，进行跨域请求。例如，www.a.com 想请求 www.b.com 的内容

//a网站的请求页：
<script>
 function jsonp(json){
    //对返回的json数据的处理
}
</script>
<script src="http;//www.b.com/jsonp.js"></script>

//b网站的jsonp.js 响应页
jsonp({'name':'amy',age:22})
