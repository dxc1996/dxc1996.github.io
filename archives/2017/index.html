3月 01, 2017 发布在 后端

public function via($notifiable)
    {
        $to = $notifiable->mobile;
        $code = mt_rand(10000,99999);
        $minutes = 5;
        // 短信内容
        $content = '【xx】亲爱的用户，您的验证码是' . $code . '。有效期为' . $minutes . '分钟，请尽快验证。';
        // 只希望使用内容方式放送,可以不设置模板id和模板data(如:云片、luosimao)
        dd(\PhpSms::make()->to($to)->content($content)->send());

        //        return ['PhpSms'];
    }

laravel-sms luosimao短息发送失败

3月 01, 2017 发布在 后端


再用laravel-sms短信扩展的时候发现lusoimao怎么发送短信返回值为

[
    {
        "driver": "Luosimao",
        "time": {
            "started_at": "0.12135700 1488351377",
            "finished_at": "0.15184700 1488351377"
        },
        "success": false,
        "result": {
            "info": false,
            "code": 0
        }
    }
]

最后发现是因为laravel-sms包里面vendor/toplan/phpsms/src/phpsms/agents/LuosimaoAgent.php里面的url地址用的https的格式 而官方给的文档则是http
